Ornith 9B (Deep Reinforce AI)

- **What's New:** This is a major breakthrough in self-scaffolding—it generates its own game plans to solve tasks instead of relying on rigid human-designed prompts.
- **Best for:** Unprecedented code generation for its size. It routinely outperforms models 40× larger, scoring 69.4 on SWE-bench Verified.
- **Where to find:** [Ornith 9B Model on Hugging Face](https://huggingface.co/collections/mindchain/small-coders-lightweight-code-generation-models) [[1](https://www.youtube.com/watch?v=UDzw3qzmtvo)]

2. North Mini Code 1.0 (Cohere)

- **What's New:** A new 30B parameter Mixture-of-Experts (MoE) model released specifically for developers.
- **Best for:** Terminal-based agentic tasks and complex software engineering workflows. It boasts a 256k context window and outscores many older 100B+ parameter models on code indices.
- **Where to find:** [North Mini Code on Hugging Face](https://huggingface.co/blog/CohereLabs/introducing-north-mini-code) [[1](https://huggingface.co/blog/CohereLabs/introducing-north-mini-code)]

3. Qwen3 & Qwen3.6-27B (Alibaba Qwen)

- **What's New:** Substantial upgrades for "Thinking Preservation" (retaining reasoning context) and repository-level reasoning.
- **Best for:** Multilingual coding agents, complex logic, and full-repository refactoring.
